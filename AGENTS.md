# Project Instructions

## Development Server

- 如果已完成改动，只需要预览改动结果，不要启动开发服务器；用户会自行启动和检查。

## Package Manager

- This is a standard Nuxt project using `pnpm`.
- Run `pnpm install` after dependency changes.
- Use `pnpm run build`, `pnpm run typecheck`, and `pnpm run lint` for validation when applicable.
